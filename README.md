# Portfolio Data Analyst & Business Intelligence Analyst

## Henrique DA COSTA

Bienvenue sur mon portfolio professionnel, réalisé dans le cadre de ma formation **Business Intelligence Analyst** chez OpenClassrooms.

Cette application a été développée avec **Python et Streamlit** afin de proposer un parcours interactif à travers mon profil, mes compétences et mes réalisations en **analyse de données** et **Business Intelligence**.

## Présentation du portfolio

Le portfolio permet de découvrir :

- mon parcours professionnel et mon profil ;
- mon CV ;
- mes principaux projets Data et Business Intelligence ;
- mes analyses réalisées avec Python, SQL, Excel et Power Query ;
- mes tableaux de bord Power BI ;
- mes notebooks, documentations et livrables de projets ;
- mes visualisations et supports de synthèse ;
- une application Data exploitant l'API Judilibre ;
- une vidéo de formation proposant une visite guidée du portfolio.

## Projets et réalisations

Le portfolio rassemble plusieurs projets couvrant les principales étapes d'une démarche Data et Business Intelligence :

- préparation, nettoyage et transformation des données ;
- analyse exploratoire ;
- interrogation et exploitation de bases de données SQL ;
- modélisation des données ;
- création d'indicateurs de performance ;
- conception de tableaux de bord interactifs ;
- data visualisation ;
- automatisation de traitements ;
- exploitation d'API ;
- restitution et communication des résultats.
